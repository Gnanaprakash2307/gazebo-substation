# Substation Simulation using Gazebo

A 3D electrical substation simulation developed using Gazebo Sim.

## Features

- Transmission Towers
- 3-Phase Busbars
- Busbar Supports
- Circuit Breakers
- Power Transformers
- Lightning Arresters
- Control Room
- Name Board
- Transformer Pads
- Perimeter Fence

## Software Used

- Ubuntu 24.04
- Gazebo Sim
- ROS 2 Jazzy

## Run

```bash
gz sim worlds/substation.sdf
